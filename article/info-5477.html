完成6个任务即可签退。

本篇攻略为任务详细介绍：



Missions

1.           亭湾骑射 / Shooting and Riding in the Bay with Pavilion

Description:

[西湖漫步] 在如今一公园附近，西湖水岸变得弯曲，水面开阔，原有一黑亭，风雨中损毁。清代雍正时期浙江总督李卫，在水域中央重建该亭，供皇帝安全地巡阅驻防的八旗子弟，此处经常刀弓竞响而鱼鸟不惊，故称“亭湾骑射”。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：西湖一公园附近

预计耗时：10min

备注信息：一公园有杭州著名景观集贤亭，白天人员稠密，禁止骑行。夜间无灯光、路线复杂、湖边无护栏，请过往人员小心。

补给点：无。

洗手间：无。

 


2.           湖滨晴雨 / Sunny and Rainy Lakeside

 

Description:

[西湖漫步] 湖滨位于西湖和城市的接壤之地，不但可以一览西湖的三面云山，更是品鉴阴晴雨雾的好地方。湖滨地区保留了许多老建筑，主要是民国以后的民居、旅馆、店铺、小别墅等。此处的音乐喷泉是游客前来杭州必去的景点之一。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：西湖音乐喷泉附近

预计耗时：10min

备注信息：音乐喷泉演出时间请自行搜索，白天人员稠密，禁止骑行。夜间请注意控制人流的围栏，请过往人员小心。

补给点：有。

洗手间：有。

 


3.           惜别白公 / Farewell Mayor Bai

 

Description:

[西湖漫步] 诗人白居易于唐朝长庆二年任杭州刺史，到任后大规模疏浚治理西湖，并筑堤建闸，灌溉农田。重修六井，深受百姓爱戴。唐长庆四年任满赴洛阳之际，出现了“杭老遮车辙”的动人送别场面。西湖边的群雕刻画了当时的场景。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：北山街与环城西路交叉口附近

预计耗时：10min

备注信息：白天人员稠密，禁止骑行。夜间无灯光、路线复杂、湖边无护栏，请过往人员小心。

补给点：无。

洗手间：无。

 


4.           断桥残雪 / Broken Bridge with Thawing Snow

 

Description:

[西湖漫步] 断桥位于西湖白堤东端，地势较高，视野开阔，是冬天观赏西湖雪景的最佳去处。在冬日阳光下，冰雪消融，石桥露出了斑驳的桥栏，而桥的两端还在皑皑白雪的覆盖下，远望去似断非断，故称断桥。民间传说《白蛇传》更为断桥增添了浪漫的色彩。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：北山街与白堤交叉口附近

预计耗时：10min

备注信息：杭州著名景点 断桥残雪，路边无法停车，白天人员稠密，无法骑行到达平湖秋月。

补给点：有。

洗手间：无。

 


5.           平湖秋月 / Autumn Moon over Tranquil Lake

 

Description:

[西湖漫步] 秋高气爽之时，西湖湖面平静如镜，皓月当空，月光与湖水交相辉映，故称“平湖秋月”。古时盛行泛舟湖上游览月景，清朝康熙皇帝巡游西湖时，御书“平湖秋月”匾额于白堤西端，从此景点固定于此，背倚孤山，面临外湖。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：浙江西湖美术馆附近

预计耗时：10min

备注信息：临近孤山、浙江省博物馆，白天人员稠密，无法骑行到达断桥残雪。

补给点：有。

洗手间：有。

 


6.           宝石流霞 / Baoshi Mount with Flying Glow

 

Description:

[西湖漫步] 宝石山位于西湖北岸，与葛岭一起组成西湖北面的天然屏障，每当朝霞或夕阳映照时，山岩呈现出五彩缤纷的迷人景色，所以得名“宝石流霞”。山上立有保俶塔，始建于五代，与西湖南岸雷峰塔遥相呼应。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：北山路葛岭附近

预计耗时：10min

备注信息：白天人员稠密，自行车与行人共用一道，西湖边无护栏请过往人员小心。

补给点：无。

洗手间：无。

 



7.           曲院风荷 / Swaying Lotus in Qu Yard

 

Description:

[西湖漫步] 曲院风荷位于西湖西侧，岳飞庙前面。南宋时，此有一座官家酿酒的作坊，取金沙涧的溪水造曲酒，闻名遐迩。附近的池塘种有菱荷，每当夏日风起，酒香荷香沁人心脾，因名“曲院风荷”。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：曲院风荷公园北门附近

预计耗时：10min

补给点：有。

洗手间：有。

 



8.           花港观鱼 / Viewing Fish on Hua Pond

 

Description:

[西湖漫步] 古时西湖后山有一条清溪流经此处注入西湖，此处称之为“花港”。南宋时，人民在此兴建花园别墅，园内栽花养鱼。康熙南巡西湖时，重新砌池养鱼，筑亭建园，立碑勒石，题有“花港观鱼”四字。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：花港观鱼东门附近

预计耗时：10min

备注信息：临近苏堤南口、游轮码头，白天人员稠密，可在任务完成后向东买票排队上湖心岛。

补给点：有。

洗手间：有。

 



9.           苏堤春晓 / Spring Dawn on Su Casuway

 

Description:

[西湖漫步] 苏堤南起南屏山，北至栖霞岭，全长近三公里，为北宋诗人苏东坡任杭州知州时，利用疏浚后湖底淤泥所建。每当春天来临，苏堤上杨柳夹岸、艳桃灼灼，晨曦初露、清风徐来，处处都是春意盎然的景象，故名“苏堤春晓”。

Tips:

开放时间：全天

起始位置：苏堤南口附近

预计耗时：10min

补给点：无。

洗手间：无。

 



10.       南屏晚钟 / Late Bell in Nanping Mount

 

Description:

[西湖漫步] 坐落于南屏山间，净慈禅寺始建于五代，距今1000多年。寺内设有一座钟楼，因钟声洪亮，再加上寺后南屏山多空穴，所以晚钟敲响，钟声穿穴回荡、传播山谷，远飘大半个杭城。康熙南巡时，听此钟声，称其“南屏晚钟”。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：雷峰塔景区附近

预计耗时：10min

补给点：无。

洗手间：无。

 



11.       雷峰夕照 / Leifeng Pagoda in Setting Sun

 

Description:

[西湖漫步] 雷峰塔坐落于净慈寺北的南屏支脉夕照峰上，亦称雷峰，始建于五代。1924年，雷峰塔因年久失修轰然倒塌，新塔于2000年重建。在雷峰塔东边的长桥公园，夕阳西下时，雷峰塔与夕阳相映成趣，是观赏西湖落日的绝佳去处。

Tips:

开放时间：全天

补给点：有。

洗手间：无。

 

 



12.       柳浪闻莺 / Orioles Singing in the Willows

 

Description:

[西湖漫步] 柳浪闻莺位于西湖东南岸。南宋时，杭州作为首都，柳浪闻莺是当时最大的御花园，称聚景园。园内有会芳殿和三堂、九亭，以及柳浪桥和学士桥。清代恢复柳浪闻莺旧景。有柳洲之名。其间黄莺飞舞，竞相啼鸣，故有“柳浪闻莺”之称。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：柳浪闻莺公园东门附近

预计耗时：10min

备注信息：摄影的好去处。白天人员稠密，禁止骑行，夜间骑行请注意大石子道路、石桥，西湖边无护栏请过往人员小心，请注意避让环湖观光电瓶车。

补给点：无。

洗手间：有。

 

 

 

 

13.       鼓楼 / Drum Tower

 

Description:

[杭城寻觅] 杭州鼓楼古代为滨海敌楼，位于吴山东麓，中山南路南段，始建于南朝。鼓楼在南宋原名朝天门，明朝更名为镇海楼。千百年间，鼓楼屡建屡毁。文革期间，鼓楼被彻底拆毁。之后，杭州市为创建“历史文化名城”，于2003年重建。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：杭州鼓楼附近

预计耗时：10min

备注信息：临近胡雪岩故居、河坊街、吴山广场、签退地点，白天人员稠密，无法骑车通过河坊街到达鼓楼，道路有小溪穿行请谨慎小心。

补给点：有。

洗手间：有。

 



14.       南宋御街 / Imperial Street of South Song Dynasty

 

Description:

[杭城寻觅] 南宋御街是南宋都城铺设的一条主要街道。史料记载，铺设南宋御街一共使用了一万多块石板。御街南起万松岭，北至武林路，是南宋临安城的中轴线。每三年皇帝都要沿着御街去往祭祖场所，同时御街两旁也集中了数万家商铺，是南宋商业娱乐中心。

Tips:

开放时间：全天

推荐交通：步行（夜间可骑车）

起始位置：南宋御街附近

预计耗时：10min

备注信息：临近河坊街、鼓楼、吴山广场、签退地点。白天人员稠密，有保安看管，禁止骑行，骑行请在晚9点后早8点前穿越。行人有过街天桥连接羊坝头，自行车不能走天桥请选择向东过十字路口到达羊坝头。

补给点：有。

洗手间：有。

 



15.       羊坝头 / Yangbatou

 

Description:

[杭城寻觅] 羊坝头位于中山中路旁。古时滨海，为防海潮侵袭在此作坝称“洋坝”，俗称“洋坝头”。宋时御街穿此而过，为杭城商业中心。明时为蔬菜鱼肉等集市贸易之地。民国时此处店铺林立，市面繁荣，沿用为区片名。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：开元路与中山中路交叉口附近

补给点：有。

洗手间：有。

 



16.       五柳巷 / Wuliu Lane

 

Description:

[杭城寻觅] 杭州市五柳巷南起斗富三桥，北至道院巷。南宋时在此建五柳园，故得名。旧时为达官贵人居住之处，后成平民栖息之地，居者多为小手工业者和商贩。清光绪三十三年(1907年)，建火车站于清泰门内，火车贯城而入，此处日渐热闹。

17.       求是书院 / Qiushi College

 

Description:

[杭城寻觅] 求是书院，浙江大学的前身，位于浙江省杭州市上城区大学路，创建于1897年，是中国近代史上效法西方学制最早创办的几所新式高等学校之一。学校由杭州知府林启利用普慈寺创办，是浙江省第一所新型高等学府。

洗手间：无。

Description:

[杭城寻觅] 莫衙营位于建国北路靠近环城北路口，古时这里是杭州的北郊。南宋时，杭州城郊驻扎有五军，莫衙营为游奕马军驻地，后变为地名。如今莫衙营附近已是今非昔比，在建的地铁五号线设站于此，一旁的东河为休闲散步的好去处。

19.       京杭大运河 / The Grand Canal

 

Description:

[杭城寻觅] 京杭大运河是世界上里程最长、工程最大的古代运河，也是最古老的运河之一，大运河南起杭州，北至北京，途经多个省份，贯通海河、黄河、淮河、长江、钱塘江五大水系，全长约1797公里。2014年被评为世界文化遗产。

Tips:

[杭城寻觅] 百井坊巷东起中山北路，西至延安路，吴越王钱镠开井九十九眼，后来多数湮塞，仅存三眼，相去各数十步。百井坊巷不仅是杭州武林商圈的核心区域，更集中了不同年代的历史建筑、历史街巷和古井，是杭州的历史缩影。

21.       贡院 / Exam Hall

 

Description:

[杭城寻觅] 贡院是封建科举制度中乡试或会试的场所。杭州贡院历经数代更迁，在1459年迁至此处，此后这里便是杭州教育的中心。1908年，改贡院为浙江两级师范学堂，后又数次更名，如今这里是杭州高级中学。

22.       孩儿巷 / Hai’er Lane

 

Description:

[杭城寻觅] 孩儿巷，位于杭州下城区，东起中山北路，西至武林路。宋称保和坊砖街巷，巷内多泥孩儿铺。南宋诗人陆游曾在这里写下了“小楼一夜听春雨，深巷明朝卖杏花。”的著名诗篇，现在的孩儿巷98号，设有陆游纪念馆。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：庆春路与中山北路交叉口附近

预计耗时：10min

备注信息：周围有高楼大厦影响，周围居民区出入口请注意避让车辆。

补给点：有。

洗手间：无。


 

23.       盐桥 / Bridge of Salt

 

Description:

[杭城寻觅] 盐桥，位于庆春路中段，桥上原有惠济庙，故原名“惠济桥”。宋朝时，在此设盐船用的榷场，因此又称“盐桥”。上世纪八十年代，整治中河时，庙宇被拆，后改建为由胡庆余堂制药厂出资的“庆余亭”，与公路桥平行，供行人行走之用。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：庆春路与中河北路交叉口附近

预计耗时：10min

备注信息：周围有高楼大厦影响，任务po略偏移主路线请注意折返，行人有过街天桥，骑行需穿越两次红绿灯，请勿逆向骑行。

补给点：无。

洗手间：无。


 


24.       龙翔桥 / Longxiang Bridge

 

Description:

[杭城寻觅] 龙翔桥位于平海路延安路路口。南宋皇帝宋理宗在谋权篡位后，把旧居改名龙翔宫。元朝时，龙翔宫遭大火焚毁，易地重建于今龙翔桥附近，一旁西河上的小桥也得名“龙翔桥”。解放后，这里曾是杭州最繁忙的公交枢纽，随着地铁开通，逐渐演变为商业中心。

Tips:

开放时间：全天

推荐交通：步行，骑行

起始位置：平海路与延安路交叉口附近

预计耗时：10min

备注信息：临近龙翔桥地铁站，周围有高楼大厦影响，白天人员稠密，请看管好自己的随身物品，需要穿行的红绿灯较多请注意安全。

补给点：有。

洗手间：有。
