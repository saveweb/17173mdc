抵抗军玩家@SheldonTan带来了适合北方玩家口味的美食攻略~

以下为正文：

----------------------------

我印象里，agents 在提起自己的家乡美食的时候，大多眉毛上挑，瞳孔扩大，一种隔着手机屏幕都看得到的兴奋。

好味道这东西本身就是一种复合的体验，那是与心情、环境、经历、习惯等等捆绑在一起的记忆。所以“亲妈做的那一碗面最好吃”，不然较起真儿来又有哪盘儿菜过得了双盲呢……

作为一个被东北、华北口味养起来的人，真到了魔都以后发现上海菜可不是仅限于传言中的甜，浓油赤酱的风格比起苏浙菜来更容易被大多数北方人接受。而魔都受地理位置和历史的影响，各地各国的美食都吃得到，并且在环境和“调调”这一项上多是有圣光加持的，若是再配上背景故事 BGM，总的体验自然不差。以下分享一些我自己比较喜欢的馆子，这些馆子也是被我的一些出差来魔都的北方损友认可和喜好的，有兴趣的 agents 可以尝试一下。（请大家自行在任何地方插入“仅代表作者口味”聊以宽心……上了岁数的agents在说话的时候身上都是贴满了免责声明的……）（所有图片地址信息来源于大众点评，一并感谢……）（以下想到哪写到哪……）（以上以下都不清真！）



黄鱼面/黄鱼煨面/雪菜黄鱼面

 

阿娘面

卢湾区思南路36号（近南昌路）

http://www.dianping.com/shop/3032710

阿娘面是一块老招牌，虽风评日下，但还可以一试（正如再怎么喷，到了帝都吃烤鸭你还是想着去全聚德试一次……）。


明呈黄鱼面馆

徐汇区斜土路1370号（近大木桥路）

http://www.dianping.com/shop/5994721

这家面馆现在有两家分店，老店在斜土路。很早吃过，印象深刻，据说后来上了电视节目。


大肠/猪肝/腰花/鳝丝面


大肠面

卢湾区复兴中路59号（吉安路东台路间）

http://www.dianping.com/shop/501534

以面的名字做馆子的名字，不多见。肠子处理得干净，香，不腻。建议拌面，双浇大肠，或者如我一个损友一样每次点“大肠大肠大肠面”


大眼面馆

黄浦区鲁班路105号（近丽园路）

http://www.dianping.com/shop/2041288

现炒的浇头，浓油赤酱，重口。大肠猪肝腰花三浇可以。有兴趣的agents可以翻翻缩头面、香葛丽的野史八卦文~


其他推荐有兰桂坊（鳝丝面（鳝丝也应是现炒）），祥和面馆（辣肉面），老地方面馆（各种面），上面坊（鸭肉面），宝泰面馆（只在过了晚上十点时候别人家不营业了他这才值得吃），味香斋（麻酱拌面、小牛咖喱汤、焖蹄）德兴馆（各种面）等等~现炒浇头的面馆一般都可以尝试，也可随便问问哪个本地agent或者附近的邻居，谁又没有个心头好呢~


牛蛙面


哈灵面馆

黄浦区广西南路8号（延安东路）

http://www.dianping.com/shop/14734396

牛蛙拌面，再加一份牛蛙浇头。没有这一碗大白腿的夜，最难将息……咸算什么……味精算什么……如果甜味、肉香、肥油这些埋在基因里对碳水、蛋白、脂肪的原罪式亲和没错，那么 umami 就是更正确的对氨基酸的渴求。我个人并不觉得花上一天一夜高汤取鲜味比现代化工带来的便捷获取谷氨酸钠有除了逼格之外的高。（化~学~就~是~你~，唱！


牛肉面


西北派的同学们此处应有很多话说，而此处很多敏感词如阿里兰面什么的……大体上敦煌楼，东方宫，面是好的。不过此处推荐的是一家台湾风味的牛肉面馆：



品味小馆/吉亨面馆

浦东新区东绣路999弄万源杰座苑8号楼103室（近东建路）

http://www.dianping.com/shop/21242195

最早吃到吉亨的半筋半肉宽面的时候激动得不行，感觉上是强于台北的永康街、桃园街及老张牛肉面的。不嫌远的同学可以去浦东的馆子，QQ 元蹄（猪蹄）更完整。



日式拉面


魔都的日式拉面也是各有各的好。此处推荐一家：


博多一幸舍

徐汇区肇嘉浜路1111号美罗城B1楼025（五番街）

http://www.dianping.com/shop/32635947

浓厚豚骨拉面，试试吧~饺子和其他也可尝试。


炸猪排


上海炸猪排，很多面馆、老式西餐馆都有，也有专门炸猪排的小摊。吃的时候配上辣酱油，满足！日式炸猪排，原来是有 Bros 会津まる家（已停业），现在想来是在旬鲜误食的其他某个 agent 先下单的猪排套餐好吃。



白斩鸡


大体上是小绍兴、振鼎鸡、泰煌鸡等几家。此处推荐：


小绍兴（云南南路店）

黄浦区云南南路69-75号（近宁海东路）

http://www.dianping.com/shop/500633

组队的 agents 可以上楼吃（正餐？），一楼就随便吃吃，边上还有大壶春~


生煎


小杨生煎连锁多，汤大，别吃~推荐大壶春，飞龙生煎，丰裕生煎

 

大壶春（云南南路店）

黄浦区云南南路89号

http://www.dianping.com/shop/57974605




小笼


别吃豫园的南翔馒头店，别吃鼎泰丰，去吃尊客来（去吃少女），此处（并不是推荐）提一个波普的八色小笼：

 

乐新皇朝（国金中心店）

浦东新区世纪大道8号国金中心二期L3-36（童装区直底）

http://www.dianping.com/shop/8959174


日本料理


魔都的日料是国内最好的这个应无争议，日式居酒屋、酒吧、料理、放题、创意菜等等一应俱全。本 agent 胃比较大，只有自助可满足……此处推荐几家日料自助：

 

初花

长宁区虹桥路1446号古北湾大酒店3楼（近古北路）

http://www.dianping.com/shop/2032762


每年十月左右到第二年二、三月份，初花的海胆是不一样的，冲着这海胆就值得一吃。其他刺身、料理也都是高分。


万岛日本料理铁板烧（静安店）

静安区乌鲁木齐北路480号万泰国际5楼（上海宾馆对面）

http://www.dianping.com/shop/6061638

“据说”万岛是脱胎于初花，但已经开了数家分店了。比起初花，万岛的铁板烧要好一些，食材上也会更新品种，鹅肝也走心（抬杠的同学请出门左转 Jean Georges 或紫外线（又，紫外线涨价了，原来错过现在就更吃不起了，cry out loud））agents也可在周五晚上吃万岛的外滩店到打烊，然后从外滩开始夜刷，吃饱了才有劲儿【握拳】

 

其他日料自助推荐荣新馆（服务员妹子给剥螯虾好评），还有最近新开的鱼家自助料理（号称上海不少日料馆子的海鲜供应商自营店，实际尝试确实新鲜）


啤酒


最后给嗜酒的北方 agents 推荐一个喝啤酒的好地方：

 

boxing cat brewery 拳击喵

长宁区遵义路122弄36号虹桥世纪电影城楼下（近虹桥南丰城）

http://www.dianping.com/shop/21146755

拳击喵在魔都有三家店，南丰城这家酒一般是很全的，店也大，不过我常去的是思南路的店……现在精酿啤酒越来越火，大家真的可以慢慢放弃美式淡拉格了，这又不是计划经济时代了……帝都的熊猫精酿酒款很多，但除了熊猫眼蜂蜜淡艾都太实验性质了……魔都的 boxing cat 的酿酒师号称拿过各路大奖（查资料似乎已于去年逝世，不远万里来到魔都的德州人民 Gary，留下了我的大爱 King Louie Imperial Stout 和 Bad Boy IPA），这里的啤酒个性鲜明而不另类，是我非常推荐的。


魔都的美食是吃不尽的，她本身就是世界美食的一个缩影，正如这个城市。大家可按喜好提前做好攻略，也可随时向本地 agents 询问，魔懒随时欢迎!（BTW, Mission Day 期间在魔都有一场郑明勋指挥的贝九，还有一场玻璃渣中毒玩家不能错过的VGL，有兴趣的 agents 请提前在各平台订票）
